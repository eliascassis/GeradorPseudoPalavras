Gerador de Pseudo-Palavras


O sistema gera pseudo-palavras a partir de outras palavras da língua portuguesa. Para gerar pseudo-palavras basta escolher as características das palavras desejadas. As características disponíveis são: canonicidade (se a palavra é canônica, i.e., possui sílabas no formato CV, ou não) e tonicidade (se a palavra é proparoxítona, paroxítona ou oxítona).
